# Burger App

## Prerequisites
- Flutter 3.7.0 (use fvm)

## Getting Started
1. Clone the project
2. Open the project in Android Studio or VSCode
3. Run the command `flutter pub get` to install dependencies
4. Run the command `flutter run` to run the app
